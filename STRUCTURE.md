# 회계던 문제 데이터 구조 정의

## standard 필드 분류 기준

questions 배열의 각 문제는 `standard` 필드로 분류됨.

| standard | 표시명 | 설명 |
|----------|--------|------|
| K-IFRS | K-IFRS | 한국채택국제회계기준. IFRS16, IAS16, IAS2 등 기준서 번호 기반 문제. 상장사/대기업 적용. |
| common | 공통 (기초회계) | 특정 기준서(IFRS/IAS 등)에 속하지 않는 재무회계 기초 문제. 기준서 번호 없이 풀 수 있는 것만 포함. 복식부기 원리, 계정 분류, 기본 거래 분개, 결산 정리, 재무제표 기초 등. |
| general | 일반기업 | 일반기업회계기준 (중소기업 적용) |
| bank | 은행업 | 은행업 회계기준. 예수부채, 대출채권, ECL, BIS비율 등 금융기관 특화 문제. |
| public | 공공기관 | 공기업·준정부기관 회계기준. 사회기반시설, 이연정부보조금, 복구충당부채 등 공공 특화 문제. |

## common 포지션 명확화

**common = 재무회계 기초/중급**

### 포함 기준 (이것만 넣어야 함)
- 복식부기 원리 (차변/대변)
- 기본 거래 분개 (현금, 외상, 어음, 비용, 수익)
- 결산 정리 분개 (선급/선수/미수/미지급)
- 재무제표 기초 (재무상태표, 손익계산서)
- 계정 분류 (자산/부채/자본/수익/비용)
- 기초 계산 (자본 등식, 매출총이익, 감가상각 기초)
- 자기주식 취득/처분
- 외화거래 기초 (환율 적용 매출/매입)
- 재고자산 저가법/감모손실
- 제조원가 기초 (원재료/재공품/제품)

### 포함 금지 (이건 K-IFRS 또는 해당 기준에 넣어야 함)
- IFRS/IAS 기준서 번호가 필요한 문제
- IFRS16 리스 (사용권자산/리스부채)
- IFRS9 금융상품 분류 (FVPL/FVOCI/AC)
- IAS37 충당부채 현재가치
- IAS19 확정급여채무
- 기타 기준서 번호 명시가 필요한 문제

## 난이도 기준

| difficulty | 기준 |
|-----------|------|
| basic | 단일 거래, 단순 분개, 계정 분류 |
| intermediate | 복합 거래, 결산 정리, 2~3단계 계산 |
| advanced | 오류수정, 마감분개, 복잡한 계산 |
| practical | 실무 맥락, 복합 판단 필요 |

| standard | 분개 | OX | 계산 |
|----------|------|-----|------|
| common | cjb(basic) / cji(intermediate) / cja(advanced) | cox(basic) / coa(advanced) | ccalc(basic) / ccalca(advanced) |
| K-IFRS | ki + 번호 | ko + 번호 | kcalc + 번호 |
| bank | bjb/bji/bja/bjp | box | bcalc |
| public | pjb/pji/pja/pjp | pob/poa | pcalc |
